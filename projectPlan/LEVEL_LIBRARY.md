# Crime Scene Cleaner — Level Library

This document contains a growing library of level scenarios.

Each level represents the **aftermath of an event**.

Levels are built using the following structure:

Level Name  
Environment  
Primary Event  
Mess Layout  
Story Props  
Hidden Clues  

---

# Episode 1 — Apartment Chaos

---

## Level 1 — Pizza Party Disaster

Environment
Apartment Living Room

Primary Event
