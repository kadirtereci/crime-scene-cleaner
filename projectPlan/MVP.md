# Crime Scene Cleaner – MVP Plan

## Goal

Create a **minimal playable prototype** of the game.

The MVP should prove that the **core cleaning mechanic is fun**.

Target session: **30–60 seconds**

---

# Core Gameplay Loop

Enter room  
↓  
Drag mop  
↓  
Clean stain  
↓  
Progress increases  
↓  
All stains cleaned  
↓  
Level complete  

---

# MVP Features

Required features

- Single room
- 5 stain objects
- 1 tool (mop)
- Drag-to-clean mechanic
- Progress bar
- Timer
- Level complete screen

Not included

- upgrades
- shops
- monetization
- multiple levels
